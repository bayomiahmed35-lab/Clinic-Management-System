# 🏥 Clinic Management System | C++

A complete console-based clinic management system built with C++.

## Features

### Admin
- Add / Remove doctors
- View all doctors and appointments
- Manage fees and discounts

### Patient
- Sign up / Login
- Book appointments (with time & date validation)
- View appointments with bill
- Cancel appointments

## Tech Stack
- C++ (No external libraries)
- Structs for data organization
- Static arrays (20 doctors, 50 patients, 100 appointments)

## How to Run

```bash
g++ clinic.cpp -o clinic
./clinic
